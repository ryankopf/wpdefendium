Defendium is a powerful spam checker plugin for WordPress, integrating seamlessly with the Defendium API to scrutinize incoming comments for spam, ensuring a clean and secure comment section.

== Description ==

Defendium is an innovative WordPress plugin designed to safeguard your website's comment section from spam. By utilizing the robust Defendium API, this plugin meticulously scans incoming comments, effectively identifying and blocking spam. It's a must-have tool for website owners seeking to maintain the integrity and cleanliness of their comment areas.

== Installation ==

1. Upload the `wpdefendium` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Defendium works out of the box. Settings can be adjusted from the WordPress 'Comments' section.

== Frequently Asked Questions ==

= What is Defendium? =

Defendium is a WordPress plugin that uses the Defendium API to check comments for spam.

= Do I need a separate API key? =

Yes, you need to sign up for a Defendium API key to use this plugin. API keys are free and can be obtained at https://defendium.com.
